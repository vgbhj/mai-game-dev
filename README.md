# Робот-исследователь

Игра на Unity 6 (URP). Робот едет по процедурно сгенерированному террейну с разными биомами (песок, лёд, камень, болото). Движение тратит энергию, причём в каждом биоме расход разный. По карте разбросаны апгрейды — если модуль подходит под биом, расход падает, если нет — растёт. Цель — добраться от старта до финиша, не потратив всю энергию.

## Управление

- W / стрелка вверх — вперёд
- S / стрелка вниз — назад
- A / стрелка влево — поворот влево
- D / стрелка вправо — поворот вправо

Апгрейды подбираются автоматически при приближении. Удалить ненужный апгрейд можно кнопкой в интерфейсе.

## Реализованные механики

- Процедурная генерация террейна (Perlin noise с октавами) с автоматической раскраской по биомам
- 4 биома (песок, лёд, камень, болото), каждый со своим множителем расхода энергии
- Система энергии — расход считается на основе текущего биома и установленных апгрейдов
- 5 видов апгрейдов с 3D-моделями и иконками — каждый модуль по-разному влияет на расход в разных биомах
- Инвентарь на 3 слота с возможностью удаления ненужных модулей
- Спавн апгрейдов по карте с привязкой к ближайшему биому
- Спавн препятствий на террейне
- Валидация проходимости уровня (BFS) — генератор гарантирует, что от старта до финиша можно дойти
- Навигация — стрелка-компас и отображение расстояния до финиша
- UI: шкала энергии, индикатор текущего биома, слоты апгрейдов
- Экраны победы и поражения с возможностью рестарта
- Камера от третьего лица с автоследованием за роботом
